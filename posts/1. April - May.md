#### Tip: Adding DWM / ST / DMENU to personal repos:

```shell
rm -rf .git   #remove the original cloned repo

git init
git remote add origin <repo link>
git remote set-url origin <repo link>
git add .
git commit -m "First Commit"
git push
```
Note that when prompt for password, use a generated Personal Access Token (with Content permission set to rw) instead.
